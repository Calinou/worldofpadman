World of Padman WoP 1.6.2 patch zip
by PadWorld Entertainment

Includes:
1. WoP 1.6.2 patch (28/11/2021)
2. WoP 1.6.1 patch (01/11/2021)
3. WoP 1.6 patch (17/12/2011)
4. WoP 1.5.1 hotfix (23/01/2011)
5. Binaries (64 bit only) for Windows, Linux, and MacOS
6. Server configuration files for all game types
7. WoP editing files (WoP mapping files.zip & WoP radiant files.zip)

The WoP 1.6.2 patch release works with any prior WoP 1.5 release
and newer, but requires at least the WoP 1.5 full version.

Instructions:
Please extract the contents of the zip file with all relative paths
into your World of Padman installation directory to update and
replace existing files. For more information about changes please
have a look at XTRAS/changelog.txt.

Your WoP Team

Please visit https://worldofpadman.net
